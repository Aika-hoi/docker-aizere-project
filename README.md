3. Понятна ли разница между образом (Image) и контейнером (Container):

Образ — это неизменяемый шаблон или "снимок" файловой системы, который содержит код, среду исполнения и библиотеки. Это пассивный объект (чертеж).
Контейнер — это запущенный и изолированный процесс, созданный на основе образа. Это активная сущность, в которой выполняется код.
